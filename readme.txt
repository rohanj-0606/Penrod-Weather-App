This is a weather application which displays weather for four cities - Chicago, Milwaukee, Dallas & Minneapolis.
This application fetches weather data for different cities from OpenWeatherMap.org API using javascript.
Lightning Design System CSS framework is used along with HTML, CSS, jQuery technologies.

Folder Structure:
1.assets
    It contains Salesforce Lightning Design System contents such as icons, images, font & styles.
    
2.css
    It contains site.css and defaults Salesforce Lightning Design System.css file.
    
3.images
    It contains the images used on the webpage.
    
4.index.html
    The actual webpage displaying the weather content.
    
5.script
    It contains the javascript file which fetches data from OpenWeatherMap API.
